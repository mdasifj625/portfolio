# Product Specification & Development Plan: Senior Backend Engineer Portfolio

## 1. Core Vision & Identity
Build a modern, production-quality personal portfolio website for Md Asif Jawed, a Senior Backend Engineer (4+ years experience). The goal is to establish high credibility, engineering depth, and production experience, targeting Engineering Managers, Recruiters, CTOs, and Founders at top-tier tech companies and high-growth startups (e.g., Razorpay, PhonePe, Swiggy, Zepto, Flipkart, Groww, Juspay, CRED).

### 1.1 Design System & Aesthetics (Production Grade)
- **Theme:** High-end, premium, minimal, recruiter-focused. Sleek Dark Mode by default, with an elegant Light Mode fallback.
- **Color Palette (Dark Mode):** 
  - Background: Deep slate/obsidian (`#0A0A0A` or `#0F172A`)
  - Surface: Subtle glassmorphism (`rgba(255, 255, 255, 0.03)`) with 1px borders (`rgba(255, 255, 255, 0.1)`).
  - Accents: Electric Indigo or Cyan (`#4F46E5` or `#06B6D4`) to signify "tech" and "backend," used sparingly for interactive elements, syntax highlighting, and hover states.
  - Text: High-contrast white for headings (`#F8FAFC`), muted slate for body text (`#94A3B8`).
- **Typography:** Modern, highly legible, and premium. 
  - Headings: **Outfit** or **Inter** (geometric, clean, authoritative).
  - Body: **Inter** or **Roboto Flex** (excellent readability for long engineering writeups).
  - Monospace (Code): **JetBrains Mono** or **Fira Code** (beautiful ligatures for code snippets).
- **Interactions:** Subtle, fluid micro-interactions using Framer Motion. Hover effects on cards (subtle lift and border glow). Avoid flashy, distracting animations; focus on smooth page transitions and scroll reveals.

## 2. Tech Stack & Infrastructure
- **Framework:** Next.js App Router (latest stable version)
- **Language:** TypeScript (Strict Mode)
- **Styling:** Tailwind CSS + custom CSS variables for themes.
- **UI Components:** shadcn/ui (radix-ui based, accessible, customizable)
- **Animations:** Framer Motion
- **Content:** MDX for Technical Deep Dives and Projects.
- **Deployment:** Vercel (CI/CD, preview environments)
- **Quality Assurance:**
  - ESLint + Prettier
  - Lighthouse Score: 95+ (Performance, Accessibility, Best Practices, SEO)
- **SEO & Metadata:**
  - Next.js Metadata API
  - Open Graph (OG) image generation for articles
  - JSON-LD structured data for the Person and Articles
  - Dynamic Sitemap & Robots.txt
